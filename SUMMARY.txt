## 🚀 Quick Start (3 Steps)

### Step 1: Build
```bash
cd build
cmake -DCMAKE_BUILD_TYPE=Release -DLLVM_DIR=$(llvm-config --cmakedir) ..
make
```

### Step 2: Run Automated Analysis
```bash
# Linux/macOS
chmod +x ../run_analysis.sh
../run_analysis.sh ../test1.ll --output-dir results --format all

# Windows
..\run_analysis.ps1 -LLFile ..\test1.ll -OutputDir results -Format all
```

### Step 3: View Results
```bash
cat results/*/report.txt              # Text report
jq . results/*/report.json            # JSON data
cat results/*/report.csv              # Spreadsheet
```

## 🔄 Typical Workflows

### Workflow 1: Find & Fix Bottlenecks
```
1. Run: ./run_analysis.sh mycode.ll
2. Read: Bottleneck section
3. Apply: Recommendation
4. Rerun: Compare improvement
```

### Workflow 2: Vectorization Opportunity
```
1. Run: analysis_toolkit.py output.txt --text
2. Find: "VECTORIZATION CANDIDATES:"
3. Apply: SIMD optimizations
4. Verify: Rerun analysis
```

### Workflow 3: Performance Regression Detection
```
1. Run: Pass on old code → save as baseline.txt
2. Run: Pass on new code → output.txt
3. Compare: ./run_analysis.sh new.ll --compare baseline.txt
4. Diagnose: Review degradation
```

### Workflow 4: Module Optimization Prioritization
```
1. Run: Pass on entire module
2. Export: JSON format
3. Parse: jq '.functions | sort_by(.total_weight_cost) | reverse'
4. Target: Optimize top 5 expensive functions
```
